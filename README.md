# 💳 A/B тест: влияние информации о кэшбеках на конверсию лендинга дебетовой карты

## 📋 Описание проекта

На лендинг с оформлением дебетовой карты была добавлена информация о дополнительных кэшбеках от партнёров. Цель — проверить, увеличивает ли это изменение конверсию из переходов на лендинг в оформление карты.

---

## 🎯 Гипотезы

- **H₀:** Конверсия в тестовой группе равна конверсии в контрольной (изменение не повлияло)
- **H₁:** Конверсия в тестовой группе выше конверсии в контрольной (изменение увеличило конверсию)

## 📊 Данные

Файл `hw_ab.csv` содержит логи переходов на лендинг за период **01.08.2023 — 01.11.2023**.

| Столбец | Описание |
|---|---|
| `date` | Дата захода пользователя на лендинг |
| `id` | Уникальный идентификатор пользователя |
| `group` | Группа: `control` или `test` |
| `converted` | 1 — оформил карту, 0 — не оформил |

**Размер данных:** 94 778 пользователей

## 🔬 Методология

### Проверка корректности теста
- Проверка уникальности пользователей
- Проверка равномерности распределения по группам
- Проверка на пропуски в датах (пропусков не обнаружено)
- Анализ динамики конверсии по дням

### Статистические тесты

Использовались два подхода:

**1. Биномиальный тест** — на основе знания о биномиальном распределении конверсий:
```python
from scipy.stats import binomtest
result = binomtest(k=conversions_control, n=n_control, p=conv_test, alternative='two-sided')
```

**2. T-тест** — на основе знания о нормальном распределении средних конверсий по дням (Центральная предельная теорема):
```python
from scipy.stats import ttest_ind
stat, p_value = ttest_ind(conv_test, conv_control)
```

## 📈 Результаты

### Размер групп

| Группа | Пользователей |
|---|---|
| Control | 47 330 |
| Test | 47 448 |

### Конверсия

| Группа | Оформили карту | Конверсия |
|---|---|---|
| Control | 4 922 | **10.40%** |
| Test | 5 674 | **11.96%** |

### Прирост конверсии
- **Абсолютный прирост:** +1.56 п.п.
- **Относительный прирост:** +15.0%
- **Отношение конверсий (lift):** 1.15

### Статистическая значимость

| Тест | Статистика | P-value | Вывод |
|---|---|---|---|
| Биномиальный тест | — | ≈ 0 | Отвергаем H₀ |
| T-тест (по дням) | 7.209 | ≈ 0 | Отвергаем H₀ |

Оба теста показали **p-value ≈ 0**, что значительно меньше стандартного уровня значимости α = 0.05.
